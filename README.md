# shop-stack
A backend interface that makes it easy to host your own shopping site.

## dependencies
- docker
- stripe account

## running
1. create `.env` file that defines `STRIPE_API_KEY` and `MYSQL_PASSWORD`
3. start the stack with `docker compose up`
